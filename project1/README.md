# 分子生成与性质预测项目

这个项目是一个简单的分子生成和性质预测系统,使用Python和RDKit库实现。

## 功能

1. 生成随机分子
2. 预测给定分子的物理化学性质

## 安装

1. 确保你已安装Python 3.7或更高版本。

2. 克隆此仓库:
   ```
   git clone git@github.com:bug-slayer/AI-Drug-Design.git
   cd molecule-generator-predictor
   ```

3. 安装所需依赖:
   ```
   pip install -r requirements.txt
   ```
